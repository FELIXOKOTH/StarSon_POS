StarSon POS

Smart, Green, and Future-Ready Point of Sale System

What is StarSon POS?

Developed by BRIGHT_ARM ENTERPRISE, StarSon POS is an eco-conscious retail management solution that tracks receipts, calculates carbon savings, and integrates with future carbon credit systems like MITI PESA.

Core Features

  • SMS and PDF receipt delivery
  • Tree-saving counter per transaction
  • Carbon credits module (hidden/future-ready)
  • Role-based cashier logins
  • Admin + Technician backend
  • Secure data sync + low-cost deployment
